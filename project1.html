Vending Machine Simulator
For this project, I was tasked with developing a Java-based application that simulates a simplified online vending machine, called Vending Machine Simulator. The main objective was to practice programming concepts in Java and build a small-scale simulation of a vending machine.
  • Object-Oriented Design
  • Encapsulation & Abstraction
  • Control Structures & Logic
  • Error Handling
