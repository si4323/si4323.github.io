Hangman Game Simulator
For this project, I built a Java-based Hangman game that simulates the classic word-guessing puzzle. The program allows a player to guess letters of a hidden word within a limited number of attempts before “losing” the game. My implementation focused on combining fundamental Java programming concepts with interactive gameplay logic.
  • String Manipulation
  • Arrays & Collections
  • Input Validation
  • Algorithmic & Problem-Solving Skills
